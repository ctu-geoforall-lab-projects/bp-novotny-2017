Soil Erosion QGIS Plugin
========================

Zásuvný plugin Soil Erosion slouží pro výpočet 
a prezentaci erozního smyvu na orné půdě.

Vstupem jsou vektorové vrstvy `Bonitovaných půdně ekologických jednotek (BPEJ)
<http://www.spucr.cz/bpej/celostatni-databaze-bpej>`_, `Registru půd (LPIS)
<http://eagri.cz/public/app/eagriapp/lpisdata/>`_ a rastrová vrstva digitálního modelu terénu.

Výstupem je rastrová vrstva znázorňující hodnotu smyvu 
.. a vektorová vrstva obsahující výsledné hodnoty průměrné 
.. dlouhodobé ztráty půdy pro definované erozně uzavřené celky.

Případné chyby a nápady na vylepšení zásuvného modulu prosím pište na `stránku
<https://github.com/ctu-geoforall-lab-projects/dp-svoboda-2017/issues>`__.
